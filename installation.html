Installation

This page explains how to install the server and the client software for Jericho Comms. Ideally the person who is setting up the server will also follow the client installation process as well to create a Portable Firefox install and give that to the other user. This will save the other user time and will be an advantage if they are not technically minded.

Server installation

This easy step by step guide covers the full process for setting up a secure server for your communications. The server side should be done first.

Client installation

This easy step by step guide covers the full process for setting up the chat program on each partner's computer.
